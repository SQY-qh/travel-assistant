## 1. 产品概述
智能旅行助手是一个以手机界面形态展示的对话式旅行规划网站，帮助用户从模糊需求到完整方案一步完成出行决策、预算规划与风险提醒。
- 面向自由行用户、家庭出游用户和首次出境旅行用户，解决需求收集分散、行程规划复杂、预算不透明、出行准备遗漏等问题。
- 产品价值在于用一个可视化、可追问、可持续更新的旅行助手，串联咨询、规划、提醒与决策支持。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通访客 | 无需注册 | 体验对话咨询、查看示例方案 |
| 旅行用户 | 本地保存会话 | 创建旅行计划、查看预算、收藏建议、设置价格监控 |

### 2.2 功能模块
1. **首页/对话页**：手机壳舞台、语音通话风格头部、对话式需求收集、快捷建议标签。
2. **旅行方案页**：目的地推荐、每日行程卡片、路线总览、预算分配、注意事项。
3. **监控与准备页**：机票酒店价格监控、签证与政策、打包清单、穿搭推荐、保险推荐。

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页/对话页 | 手机界面舞台 | 桌面端居中展示一台手机，内部为旅行助手主界面，模拟语音助手式交互体验。 |
| 首页/对话页 | 对话式需求收集 | 通过多轮问答收集目的地意向、出发地、日期、同行人、预算、旅行偏好、签证状态、住宿标准、出行节奏。 |
| 首页/对话页 | 快捷意图标签 | 提供“帮我去海边放松”“3 天城市漫游”“亲子轻松游”等快捷按钮，一键填充需求。 |
| 首页/对话页 | 实时需求摘要 | 在对话旁以摘要卡形式动态更新已收集信息，提示还缺失哪些关键字段。 |
| 旅行方案页 | 目的地/行程推荐 | 基于用户偏好给出 3 组候选目的地和推荐理由，支持切换查看。 |
| 旅行方案页 | 行程路线生成 | 生成按天组织的景点、餐饮、交通和时间安排，展示地图路线、城市动线和日程节奏。 |
| 旅行方案页 | 预算制定与分配 | 按交通、住宿、餐饮、门票、保险、机动费用给出预算总览、占比图和建议调整。 |
| 旅行方案页 | 政策与注意事项 | 根据出发地和目的地输出签证、入境政策、天气、支付、通信、治安、文化礼仪等提示。 |
| 监控与准备页 | 机票/酒店价格监控 | 录入目标价格与日期区间，展示近期价格走势、提醒建议与监控状态。 |
| 监控与准备页 | 打包清单与穿搭推荐 | 基于天气、时长和活动场景生成携带清单、穿搭建议和缺失物品提醒。 |
| 监控与准备页 | 保险推荐 | 按出境/境内、冒险程度、老人儿童同行等条件推荐合适保险类型与要点。 |

## 3. 核心流程
用户进入首页后，先以语音助手式界面与系统对话。系统通过多轮追问收集需求，并实时汇总已知信息。当信息足够时，系统生成候选目的地与完整旅行方案。用户可以继续追问和微调，随后切换查看路线、预算、价格监控、签证政策、打包清单和保险建议。整个流程保持对话入口统一，结果以卡片式模块展开。

```mermaid
flowchart TD
    A["进入网站"] --> B["查看手机界面欢迎屏"]
    B --> C["发起对话或点击快捷需求"]
    C --> D["系统多轮收集旅行需求"]
    D --> E["生成需求摘要并判断信息是否完整"]
    E -->| "不完整" | D
    E -->| "完整" | F["生成目的地候选与推荐理由"]
    F --> G["输出每日行程与路线"]
    G --> H["计算预算并进行分配"]
    H --> I["展示签证政策与注意事项"]
    I --> J["生成价格监控、打包清单、穿搭与保险建议"]
    J --> K["用户继续追问并调整方案"]
```

## 4. 用户界面设计
### 4.1 设计风格
- 主色调：奶油白、暖金黄、石墨灰，局部使用深红作为关键操作强调色。
- 按钮风格：圆角胶囊按钮与悬浮圆形主按钮结合，强调手机系统级交互质感。
- 字体建议：标题使用较具识别度的中文展示字体，正文使用清晰的人文无衬线字体，模拟高端消费级应用观感。
- 布局风格：桌面端为“设备展示台 + 手机屏幕”组合，手机内部为单列纵向卡片流。
- 图标建议：使用线性图标、微拟物按钮和暖色插画角色，营造会说话的旅行伙伴感。

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 首页/对话页 | 欢迎头部 | 状态栏、通话标题、动态声波、虚拟助手头像、暖色光晕背景。 |
| 首页/对话页 | 对话区域 | 左右气泡、输入框、快捷标签、语音按钮、动态打字与加载状态。 |
| 首页/对话页 | 助手角色区 | 中央 3D/插画风角色、呼吸动效、聆听态波纹。 |
| 旅行方案页 | 方案概览 | 目的地封面图、推荐指数、最佳时段、旅行风格标签、核心亮点。 |
| 旅行方案页 | 每日行程 | 时间轴卡片、景点模块、餐饮建议、交通方式、地图预览。 |
| 旅行方案页 | 预算模块 | 预算总额卡、环形占比图、分类明细、超支提示和替代建议。 |
| 监控与准备页 | 监控模块 | 价格趋势小图、监控开关、目标价格、波动标签。 |
| 监控与准备页 | 政策与准备模块 | 签证政策卡、入境提醒、天气卡、穿搭卡、打包 checklist、保险卡。 |

### 4.3 响应式策略
- 以手机界面为核心视觉，桌面端优先展示完整设备舞台与说明信息。
- 平板端保留设备轮廓并缩小辅助说明区。
- 移动端去掉外部设备舞台，仅保留核心界面内容，保证真实可用。
- 所有触控区域按移动端标准优化，支持滑动、点击、折叠展开和底部操作栏。
